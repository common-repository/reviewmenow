Show Google Places Reviews on your WordPress website who are registered with http://reviewmenow.com

== Description ==

This plugin displays Google Business Reviews on your websites using shortcode.

plugin only works for clients who are registered with http://reviewmenow.com

* Support shortcode
* Display up to 5 Google business reviews.
* Display rating

== Installation ==

1. Download and upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the Settings->Review Me Now and save Google Place Id.
4. Use Shortcode [reviewmenow] in page.
